# コンポーネントの使い方

## コンポーネントの分割
### コンポーネント化の流れ
1. コンポーネントを関数で作成する
2. コンポーネントをエクスポートする
3. コンポーネントを読み込む

1. srcフォルダ下にApp.jsを作成する
~~~App.js
  const App = () => {
    return (
      <>
        <h1>こんにちは</h1>
        <p>お元気ですか?</p>
      </>
    );
  };
~~~

2. 他のファイルでも使えるようにexportする必要がある
~~~App.js
export App = () => {
    return (
      <>
        <h1>こんにちは</h1>
        <p>お元気ですか?</p>
      </>
    );
  };
~~~

3. App関数のコンポーネントを読み込む
~~~index.js
import ReactDOM from "react-dom";s
import {App} from "./App";
~~~

コンポーネントファイルの拡張子は.jsxにする
